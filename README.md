# segundo-parcial-lp1

Este repositorio contiene la resolución del Segundo Parcial de la materia Lenguajes de Programación 1, desarrollado en el lenguaje R. 
El objetivo es analizar datos reales de comportamiento de clientes en shoppings (dataset de Kaggle) utilizando solo base R, sin librerías externas.

El trabajo incluye:

1. Lectura y almacenamiento de datos desde un archivo CSV.
2. Agrupamiento de edades en rangos definidos.
3. Clasificación de usuarios por cantidad de compras realizadas.
4. Conteo de ítems comprados por tipo y talle.
5. Análisis de colores de los productos comprados.

# Ejecución
1. Abrir la carpeta en RStudio.  
2. Cargar el archivo `shopping_behavior_updated.csv`.  
3. Ejecutar todo el script `parcial2.R`.  
4. Los resultados aparecen en consola.
